HTML

Linguagem básica para criação de WebSites.

Todo site que abrimos dentro de navegador é mostrado em sua forma final para o usuário, ou seja, a aparência do site em si. Até mesmo esse site que você está visualizando agora é uma interpretação que o navegador está fazendo de uma programação previamente feita, exibindo de forma compreensível ao usuário as várias linhas de código que foram produzidas utilizando uma linguagem de programação, a linguagem HTML.

Baseada em marcações, o HTML possui diversas “tags” que designam determinadas partes do site. Veja algumas das mais importantes linhas:

<H1> - É utilizada para demarcar um título.
Exemplo:
<h1> texto do título</h1>
<P>- Inicia um novo parágrafo
Exemplo:
<p>texto do parágrafo.</p>

*Normalmente, parágrafos possuem diversos caracteres, letras menores que as do título
*Pode-se observar que as marcações são abertas e fechadas. Geralmente, fecha-se uma marcação com a “/”.

Estruturas Básica

Geralmente, a estrutura do HTML vai ser a mesma, no caso, esta:

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="utf-8">
<title>Título da página</title>
</head>
<body>
... aqui vai todo o código HTML do site...
</body>
</html>

Não se assuste com todas essas marcações, todas são bem simples. Confira:

!DOCTYPE – Avisa os navegadores sobre o tipo de arquivo que será carregado.
HTML – Mostra que toda a seguinte página estará na linguagem HTML.
Lang – Designa a linguagem da página.
Head – Onde ficam localizadas as principais informações do site, como o título e a linguagem.
Meta Charset – Define a codificação utilizada pela página (no caso, o alfabeto utilizado).
Body – Corpo do site, onde se localizam as principais partes de um site.
